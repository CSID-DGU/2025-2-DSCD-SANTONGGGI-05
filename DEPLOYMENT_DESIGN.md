## 1. 프로젝트 개요

### 1.1 서비스 소개

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    AI Shopping Assistant                                 │
```

### 1.2 핵심 기능

| 기능 | 설명 | 기술 요소 |
|------|------|----------|
| **AI 채팅** | 자연어 기반 상품 검색 및 추천 | OpenAI GPT-5, MCP Protocol |
| **상품 검색** | 11번가, 네이버 실시간 검색 | External API Integration |
| **맞춤 추천** | 구매 이력 기반 협업 필터링 | Pandas, PostgreSQL |
| **장바구니** | 다중 플랫폼 상품 통합 관리 | SQLAlchemy ORM |
| **소비 통계** | 구매 패턴 시각화 대시보드 | Nivo Charts |

### 8.2 예시 워크플로우 파일

```yaml
# .github/workflows/deploy.yml
name: CI/CD Pipeline

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

env:
  AWS_REGION: ap-northeast-2
  ECR_REGISTRY: ${{ secrets.AWS_ACCOUNT_ID }}.dkr.ecr.ap-northeast-2.amazonaws.com

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Frontend Test
        working-directory: ./frontend
        run: |
          npm ci
          npm run lint
          npm run type-check

      - name: Backend Test
        working-directory: ./backend
        run: |
          pip install -r requirements.txt
          pytest tests/

  build:
    needs: test
    runs-on: ubuntu-latest
    if: github.ref == 'refs/heads/main'
    steps:
      - uses: actions/checkout@v4

      - name: Configure AWS
        uses: aws-actions/configure-aws-credentials@v4
        with:
          aws-access-key-id: ${{ secrets.AWS_ACCESS_KEY_ID }}
          aws-secret-access-key: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
          aws-region: ${{ env.AWS_REGION }}

      - name: Login to ECR
        run: |
          aws ecr get-login-password | docker login --username AWS --password-stdin $ECR_REGISTRY

      - name: Build & Push Images
        run: |
          docker build -t $ECR_REGISTRY/backend:${{ github.sha }} ./backend
          docker push $ECR_REGISTRY/backend:${{ github.sha }}
          docker tag $ECR_REGISTRY/backend:${{ github.sha }} $ECR_REGISTRY/backend:latest
          docker push $ECR_REGISTRY/backend:latest

  deploy:
    needs: build
    runs-on: ubuntu-latest
    steps:
      - name: Deploy to EC2
        uses: appleboy/ssh-action@v1.0.0
        with:
          host: ${{ secrets.EC2_HOST }}
          username: ec2-user
          key: ${{ secrets.EC2_SSH_KEY }}
          script: |
            cd /home/ec2-user/app
            docker-compose pull
            docker-compose up -d --remove-orphans
            docker image prune -f
```

## 부록

### A. 명령어 참조

```bash
# 개발 환경 실행
cd frontend && npm run dev
cd backend && uvicorn app.main:app --reload --port 8000

# 프로덕션 빌드
cd frontend && npm run build
docker-compose build

# 배포
docker-compose pull && docker-compose up -d

# 로그 확인
docker-compose logs -f backend
docker-compose logs -f --tail=100 nginx

# 상태 확인
docker-compose ps
curl http://localhost/health | jq .

# DB 마이그레이션
docker-compose exec backend alembic upgrade head
docker-compose exec backend alembic downgrade -1

# 롤백
docker-compose down
docker-compose up -d --force-recreate
```

### B. 트러블슈팅

| 증상 | 원인 | 해결 방법 |
|------|------|----------|
| 502 Bad Gateway | Backend 미실행 | `docker-compose logs backend` 확인 |
| DB 연결 실패 | PostgreSQL 미실행 | `docker-compose up -d postgres` |
| MCP 타임아웃 | 네트워크 분리 | Docker network 확인 |
| OpenAI 오류 | API 키 만료 | SSM Parameter 업데이트 |
| 높은 지연시간 | 리소스 부족 | EC2 인스턴스 업그레이드 |
